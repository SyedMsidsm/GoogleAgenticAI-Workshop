# Multi-Agent AI Workshop Project

This project was developed during a Google Developer Groups (GDG) workshop on building AI applications with Google's Agent Development Kit (ADK) and Gemini API.

## 🚀 Overview

This repository demonstrates a multi-agent AI system where specialized agents collaborate to complete complex tasks through structured workflows and intelligent task delegation. The project explores modern AI engineering concepts such as agent orchestration, structured outputs, memory management, and agent-to-agent communication.

## ✨ Features

- Multi-agent architecture using Google ADK
- Agent-to-agent communication
- Sequential workflow execution
- Structured outputs with Pydantic
- Gemini-powered reasoning and content generation
- Intelligent task routing and delegation
- Modular and extensible design

## 🛠️ Tech Stack

- Python
- Google Agent Development Kit (ADK)
- Gemini API
- Pydantic

## 📚 What I Learned

- Designing multi-agent AI systems
- Building agent workflows and pipelines
- LLM orchestration techniques
- Structured data generation
- Agent communication patterns
- Developing production-oriented AI applications

## 🎯 Project Goal

To gain hands-on experience with Google's ADK ecosystem and understand how multiple AI agents can work together to solve real-world problems more effectively than a single agent.
